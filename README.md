fresh harvest is a full stack web application that allows user to brouse for vegetables and fruits in a simple and organised way
Tech stack:
*PHP
*MYSQL
*Javascript
*CSS

Features:
*Browse products
*Category filtering
*Organized product display

How to run locally:
*Install XAMPP ,start MYSQL and Apache
*place project in htdocs
*create database name as fresh_harvest .click import button and import database in phpMyAdmin which is located inside database folder
*run the project in any browser ,type localhost/fresh-harvest and enter
